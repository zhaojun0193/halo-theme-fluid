## 安装
主题仓库地址 https://github.com/zhaojun0193/halo-theme-fluid
在 Release 页面下载最新版本 zip 后，通过 Halo Console 安装上传即可。

## 已实现功能
- [x] 文章置顶
- [x] 自定义背景图
- [x] 文章目录
- [x] 代码高亮
- [x] 主题模式（深色/浅色）
- [x] 评论系统
- [x] 数学公式
